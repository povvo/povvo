# Povvo Canonical Showcase

This agent skill includes two compact visual anchors for design-language inspection:

- `assets/povvo-logo-design-language-banner.png`: canonical geometry, typography, palette, layout, texture, and scan-edge reference.
- `assets/povvo-p-logo.png`: canonical interrupted-aperture `P` reference.

## Language

Warm off-white and ink black dominate. Pale cyan is restricted to ruler ticks, registration marks, construction outlines, nodes, and scan edges. Paper grain and optical edge falloff stay restrained. Only `POVVO` may appear as readable text.

Use `../DESIGN.md` and `../outputs/prompt-pack.md` as the production contract. Do not introduce substitute marks, additional logo constructions, product or lifestyle subjects, people, sport imagery, saturated blue surfaces, glossy technical materials, or generic futuristic interfaces.
